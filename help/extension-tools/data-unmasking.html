Data Unmasking


Data Unmasking is a simple tool to unmask modules of your current work according to the mask pattern that being used. It can be used when you want to analyze QR code module by module manually.

This tool is toggleable like Brute-force Format Info Pattern. Activating this tool will add new toolbox under main toolbox in Editor mode where you can manage the tool settings there.

1. Mask Pattern : Set the mask pattern that will be used to unmask QR code (change this will affect current Format info pattern).

2. Mask Pattern Area : Option whether to show Mask pattern area or not (the affected modules will be marked with green color).

3. Unmask Data : Toggle between unmask/mask data of QR code.
